# How often do you update the Agility World Index?

**URL:** https://help.comparativeagility.com/3-general/how-often-do-you-update-the-agility-world-index/
**Category:** General

---

The Index is updated every single day as more people use the tool. However, because the Index is based on a large set of data, there are few month-to-month changes in the AWI.

If you’re looking for a more “dynamic” benchmark, we recommend two things:

  * Leverage the **industry-specific benchmark**. These benchmarks are smaller in size so you’ll be able to see the changes reflected more quickly. They should also be more relevant to your specific situation, given that there are often significant changes in agility from industry to industry.
  * While it makes sense to use the **AWI** as an initial benchmark as you start your journey, we recommend using your own company as the benchmark going forward. That is, when you start, you can quickly see how you’re doing compared to the world at large, but that’s not necessarily a goal.



Measuring yourself against yourself at a different point in time is an extremely effective continuous improvement strategy. In essence, it’s what agility is all about: being more agile today than you were yesterday.

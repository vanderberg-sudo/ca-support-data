# I want to access to all features. What is a good plan to get started?

**URL:** https://help.comparativeagility.com/3-general/i-want-to-access-to-all-features-what-is-a-good-plan-to-get-started/
**Category:** General

---

We would recommend starting with the [Premium](https://www.comparativeagility.com/pricing) plan if you’re in a corporation with more than a few teams. It will allow you to customize the survey by adding logo/branding, select dimensions, and additional security features. Also, it will be easy to upgrade your plan once you need more collectors.
